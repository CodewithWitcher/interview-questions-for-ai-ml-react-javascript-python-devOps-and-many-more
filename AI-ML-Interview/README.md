# AI / ML Interview Preparation

A comprehensive collection of interview resources for **AI/ML Engineer** roles — covering LLMs, Transformers, RAG, Prompt Engineering, Python for AI, and Data Science fundamentals.

---

## 📂 Contents

### 1. [LLM and Transformers](LLM%20and%20Transformers/)
- **117 Q&A** across 39 files covering architecture, attention mechanisms, fine-tuning, tokenization, alignment, and deployment.
- Files: `QA_1-3.md` through `QA_115-117.md`

### 2. [RAG (Retrieval-Augmented Generation)](RAG/)
- **105 Q&A** across 35 files covering vector databases, embedding models, chunking strategies, retrieval pipelines, and evaluation.
- Files: `QA_1-3.md` through `QA_103-105.md`

### 3. [Prompting Techniques](Prompting%20Techniques/)
- **29 techniques** with detailed explanations and examples.
- **Basic Techniques:** Zero-Shot, Few-Shot, Role Prompting, Emotion Prompting, Batch Prompting
- **Advanced Techniques:** Chain-of-Thought (CoT), Self-Consistency, Tree-of-Thoughts, Meta Prompting, Plan-and-Solve, Self-Refine, Step-Back, and 12+ more

### 4. [50 Advanced Python Concepts for AI Engineers](50%20advanced%20Python%20concepts%20for%20AI%20Engineers/)
- **54 Python scripts** covering arrays, GIL, multithreading, multiprocessing, metaclasses, asyncio, decorators, dataclasses, memory profiling, descriptors, and more.
- Includes a [Theory Explanation](50%20advanced%20Python%20concepts%20for%20AI%20Engineers/Theory%20Explanation/) sub-folder

### 5. [Data Science — iNeuron](Data%20Science%20(iNeuron)/)
- **30 PDF resources** covering core data science topics, statistics, and machine learning foundations.

### 6. [Other Readables](Other%20Readables/)
- **30 Agentic AI Interview Questions and Answers** (PDF)
- **The Ultimate Generative AI Architect Interview Q&A Guide** (PDF)

---

## 🎯 How to Use

1. Pick a topic folder based on the role you are preparing for.
2. Work through the Q&A files sequentially or jump to specific topics.
3. Practice the Python code examples hands-on for retention.
